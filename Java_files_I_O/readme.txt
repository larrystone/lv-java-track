* From terminal
- cd into Java_files_I_O from project's root folder
 `cd Java_files_I_O`
- run `javac CopyFileContent.java` to compile the source code
- run `java CopyFileContent` to run the bytecode
- Verify that destination.txt file is created in the directory and it contains the same content as soure.txt

* From IDE
- Right click on the CopyFileContent.java file and click run
- Verify that destination.txt file is created in the directory and it contains the same content as soure.txt
