* From terminal
- cd into Exceptions_in_java from project's root folder
 `cd Exceptions_in_java`
- run `javac NumberInputValidator.java` to compile the source code
- run `java NumberInputValidator` to run the bytecode
- Try out with Numerical and Non-numerical inputs

* From IDE
- Right click on the SimpleCalc.java file and click run
- Try out with Numerical and Non-numerical inputs
